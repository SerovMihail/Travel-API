Все эти стили были установлены с помощью workWithDom Framework v.0.0.1

Что может этот фрейморк?

  • Найти элемент по: id, name, tag, class, elem, allElem. Используйте метод findBy(type, elem)
  • Найти элемент используя XPath. Используйте метод findByX(path)
  • Добавить класс. addClass(newClass) или удалить класс removeClass(deleteClass)
  • Получить стили css. getCSS(type)
  • Установить стиль css. setCSS(type, value)
  • Используйте привычный синтаксис с использованием цепочек цепочек вызовов (chaining)
  • Пример использования с XPath: wwd.findByX('div[1]/div[3]').addClass('col-md-4').setCSS('height', '300px').setCSS('background-color', 'yellow')
  • Пример использования без XPath: wwd.findBy('allElem', 'li').addClass('list-group-item')
